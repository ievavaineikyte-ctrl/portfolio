## How to View This Project

### Run locally

```bash
git clone https://github.com/ievavaineikyte-ctrl/portfolio.git
cd portfolio
npm install
npm run dev
```

Open **http://localhost:5173** in your browser.

> Do **not** open `index.html` directly from the file system — this is a React + Vite project and requires the dev server or a production build.

### Production build

```bash
npm run build
npm run preview
```
