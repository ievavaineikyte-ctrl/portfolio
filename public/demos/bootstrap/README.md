## How to View This Project

### Option 1 — GitHub Pages (recommended for sharing)

1. Go to **Settings → Pages**
2. Under **Build and deployment**, set Source to **Deploy from a branch**
3. Branch: `main` | Folder: `/ (root)`
4. Save — your site will be live at:
   ```
   https://ievavaineikyte-ctrl.github.io/bootstrap-portfolio/
   ```

### Option 2 — Open locally

1. Clone or download this repository
2. Open **`index.html`** in your browser (Chrome, Edge, or Firefox)
3. Do **not** open other HTML files first — `index.html` is the entry point

```bash
git clone https://github.com/ievavaineikyte-ctrl/bootstrap-portfolio.git
cd bootstrap-portfolio
# Then double-click index.html, or run a local server:
python -m http.server 8080
# Visit http://localhost:8080
```
